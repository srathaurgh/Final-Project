Cat Meme Generator

The Cat Meme Generator generates cats based off the emotion the user chooses. There are four types of emotions the user can choose from; happy, sad, mad, and surprised. Based of the input the user will be presented with a cat meme and if emotion button is reclicked, the user will be presented of another meme relavent to the same emotion.
